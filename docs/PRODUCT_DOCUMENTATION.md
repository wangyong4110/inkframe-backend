## 📖 产品概述

InkFrame 是一个基于 AI 的智能小说创作平台，帮助用户从零开始创作高质量的中长篇小说，并自动生成配套的视频内容。系统通过多层次的质量控制和一致性保证，确保创作的连贯性和专业水准。

### 核心价值

- ✨ **零门槛创作**：从创意到成品，AI 全程辅助
- 📚 **知识管理**：自动构建世界观、角色、剧情知识库
- 🎬 **一健成片**：从小说到视频，全自动流程
- 🤖 **多模型支持**：灵活选择 AI 模型，优化质量和成本
- 🎯 **人工掌控**：每个环节都可介入、修改、优化
- 🔄 **持续优化**：A/B 测试，对比效果，选择最佳方案

### 目标用户

- 📝 **网络小说作者**：快速产出高质量内容
- 🎨 **内容创作者**：制作小说改编的视频
- 🎭 **剧本编剧**：开发剧本和故事线
- 📖 **自媒体博主**：批量创作内容
- 🎬 **视频制作者**：快速制作叙事视频

## 🎭 用户故事

### 故事 1：网络小说作者快速创作

**作为**一名网络小说作者，  
**我想要**快速创作一部高质量的玄幻小说，  
**以便**在连载平台上吸引读者。

**场景流程**：

1. 📝 **创意输入**：输入一个初始创意"一个现代程序员穿越到修真世界"
2. 🧠 **AI 辅助**：系统生成世界观设定、主角设定、大纲
3. ✏️ **人工优化**：我修改和优化 AI 生成的设定
4. 📖 **章节创作**：AI 生成章节内容，我进行润色和调整
5. 🎬 **视频生成**：选择重点章节，生成短视频进行推广
6. 📊 **数据分析**：查看章节阅读数据，调整创作方向

**人工介入点**：
- ✏️ 修改世界观设定
- 👥 调整角色设定
- 📝 编辑章节内容
- 🎬 选择视频风格
- 📊 分析读者反馈

## 🔌 MCP 工具管理

### 功能概述

MCP（Model Context Protocol）工具管理页面位于 `/model` 的「MCP 工具」标签页，支持注册和管理外部工具服务，并将其绑定到指定 AI 模型。

### 核心操作

#### 注册工具

| 字段 | 说明 |
|------|------|
| 工具名称 | 全局唯一标识符（英文+下划线）|
| 显示名称 | 界面展示名称 |
| 描述 | 工具功能说明 |
| 传输类型 | http / sse / stdio |
| Endpoint | 工具服务地址（http/sse 必填）|
| Headers | 请求头 JSON（如鉴权 Token）|
| 超时时间 | 毫秒（存储为秒，前端自动换算）|

#### 连通性测试

点击「测试连接」按钮，系统对 http/sse 工具发送探测请求，显示：
- 连接状态（正常 / 异常）
- 响应延迟（ms）
- 错误信息（如有）

#### 工具与模型绑定

在模型详情侧边栏，可为该模型勾选/取消 MCP 工具。绑定后，使用该模型进行生成时，AI 可调用已绑定的工具能力。

### 用户流程

```
注册 MCP 工具 → 测试连通性 → 在模型页面绑定 → 使用该模型生成内容时调用工具
```

## 🎬 视频分镜管理页

### 页面入口

`/video/:id` — 进入视频的分镜脚本管理页面。

### 质量档位选择

创建视频时选择质量档位，决定后续生成的分辨率和成本：

| 档位 | 标签 | 适用场景 |
|------|------|---------|
| `draft` | 草稿 | 快速验证分镜结构 |
| `preview` | 预览 | 日常创作（默认）|
| `final` | 正式 | 最终交付 |

### 镜头操作

#### 切换生成模式

每个镜头可独立切换「静态图像」或「AI 视频」模式：
- **静态**：生成图像 + Ken Burns 动效，成本低，速度快
- **视频**：AI 生成真实动态视频，效果更好，成本较高

#### 单镜头生成

点击单个镜头的「生成」按钮，异步触发该镜头的图像/视频生成。

#### 批量生成

勾选多个待生成镜头 → 点击「批量生成」，一次提交多个生成任务。

### 镜头状态

| 状态 | 颜色 | 含义 |
|------|------|------|
| `pending` | 灰色 | 待生成 |
| `generating` | 黄色 | 生成中 |
| `completed` | 绿色 | 已完成 |
| `failed` | 红色 | 生成失败 |

## ✨ 前端 UX 增强

### 全局操作反馈（Toast 通知）

所有关键操作（保存、生成、删除等）完成后，右上角弹出 Toast 通知：

| 类型 | 颜色 | 显示时长 |
|------|------|---------|
| 成功 | 绿色 | 3 秒 |
| 错误 | 红色 | 5 秒 |
| 警告 | 黄色 | 3 秒 |
| 信息 | 蓝色 | 3 秒 |

支持手动点击关闭按钮提前消除。

### 删除确认对话框

所有删除操作（删除项目、删除章节、删除角色、解绑工具等）弹出确认对话框：
- 标题 + 危险提示描述
- 危险操作确认按钮为红色
- 支持 Escape 键取消

### 章节编辑器增强

| 功能 | 触发方式 |
|------|---------|
| 自动保存 | 内容变化后 2 秒防抖触发，状态栏显示「已自动保存 HH:mm」|
| 快捷键保存 | `Ctrl+S` / `Cmd+S` |
| 离开保护 | 有未保存内容时，路由跳转/浏览器关闭弹出确认 |

### 请求超时保护

所有 API 请求内置 120 秒自动超时（AbortController），超时后：
1. 中止请求
2. 弹出红色 Toast 提示「请求超时，请重试」
